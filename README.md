**Core Principles:**
- **Layered & Modular** — AI, calculator, and content engines are independently deployable
- **Zero-disruption** — Existing calculators work perfectly without AI (AI is additive)

- **Privacy-first** — All personalization data stays in localStorage/IndexedDB by default
- **Graceful degradation** — If AI is offline, calculators fall back to deterministic formulas

## 🧠 AI Agent System

FinCalc India uses a **multi-agent architecture** — specialized AI agents that each handle a domain with deep expertise, orchestrated by a central router.

| Agent | Domain | Capabilities |
|:--|:--|:--|
| **🧾 Tax Agent** | Income Tax, GST, ITR | Regime comparison · Section 87A optimization · HRA strategy · Capital gains planning · ITR form recommendation · Deduction discovery |
| **📈 Investment Agent** | SIP, MF, Gold, FD, Stocks | Portfolio allocation · Risk-adjusted returns · Goal-based planning · SIP vs lump-sum decision · Asset rebalancing · ELSS vs non-ELSS |
| **⚠️ Risk Agent** | Insurance, Emergency, Debt | HLV calculation · Term insurance sizing · Emergency fund gap · Debt-to-income analysis · Credit risk scoring · Health cover recommendation |
| **🏦 Loan Optimizer** | EMI, CIBIL, Eligibility | Best bank selection · Tenure optimization · Prepayment strategy · CIBIL improvement path · Balance transfer savings · FOIR analysis |

### Agent Orchestrator

A lightweight router dispatches user queries to the right agent(s). Complex questions ("Should I prepay my home loan or start SIP?") trigger **multi-agent collaboration** — Loan Optimizer + Investment Agent reason together, returning a unified recommendation.

```
User: "Bangalore mein ₹60K basic hai, kya main HRA claim karun ya home loan?"
      │
      ▼
[Orchestrator] → Tax Agent (HRA calc) + Loan Optimizer (home loan interest)
      │
      ▼
Unified recommendation with:
  • Tax saving under each option
  • Break-even analysis
  • 5-year projection
  • Explainable reasoning
```

## 🧬 Digital Financial Twin

One of the most ambitious features in v4.0 — a **living AI clone of the user's finances** that continuously learns, predicts, and advises.

**What the Twin does:**
- Builds a model of income, spending patterns, savings rate, risk behavior
- **Predictive behavior tracking** — "You're likely to overspend next month based on historical patterns"
- **Spending insights** — categorizes, benchmarks against peers anonymously
- **Saving insights** — identifies leakage, suggests auto-save rules
- **Shadow simulations** — runs "what-if" in background on every new calculation
- **Drift detection** — flags when actual behavior diverges from goals

**Privacy architecture:**
- Twin lives entirely on-device (IndexedDB) by default
- Optional encrypted cloud sync for multi-device continuity
- Zero PII ever sent to any LLM — only abstracted patterns
- User can export, delete, or reset the Twin at any time

**Example Twin insights:**
```
🔔 Your Twin noticed:
  • Savings rate dropped from 22% → 14% in last 3 months
  • Likely cause: increased dining-out (3x normal)
  • Impact on retirement goal: delay of ~8 months
  • Suggested action: set ₹3,000/month auto-SIP to recover
```
